===============

Before starting to develop applications in PCNsim, check if you meet all the :doc:`prerequisites` and programs installed on your device. You'll not be able to run PCNsim without the described requisites.

After installing all prerequisites, you can follow to :doc:`testing` and start working with PCNsim on your local device.
